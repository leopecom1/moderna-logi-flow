# Moderna Logi-Flow

Sistema de gestión logística integral para Moderna Hogar.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, DB, Realtime, Edge Functions)
- TanStack React Query

## Desarrollo local

```sh
npm install
npm run dev
```

## Deploy

Desplegado en Coolify con Docker (nginx + multi-stage build).
